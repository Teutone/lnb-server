Test the api

This page has a utility function embedded in the page. Open the console (F12) and try it:

g('GET', 'tracks').then(data => console.log(data));

If there are no users yet, you can set the first admin user by hand:

g('POST', 'install', { 'username': ADMIN_USERNAME, 'password': YOUR_PASSWORD }).then(data => console.log(data));
